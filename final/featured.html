Featured Project

Camp Crystal Lake Design

Recreation of Figma Layout Design using HTML and CSS

I am most proud of this Project because this is the one I overcame many struggles with. Learning to align every element onto the page to recreate the Figma layout was a challenge. The requirements of this assignment included using a CSS grid for the overall layout and the use of flex boxes to position the elements. The fonts were used from Google Fonts specified from the Figma Design.
